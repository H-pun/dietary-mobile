# Dietary

Dietary is a comprehensive dietary tracking app designed to help users manage their daily nutritional intake, monitor their dietary habits, and achieve their health goals. The app offers a variety of features to enhance the user experience and provide valuable insights into their diet and nutrition.

# Tools & Libraries

- **Room**: Efficient database management with Room for storing user data.
- **ViewModel**: Ensures data persistence and lifecycle awareness.
- **Coroutine**: Provides seamless asynchronous programming.
- **Jetpack Compose**: Modern UI toolkit for building native interfaces.
- **Ktor**: Networking library for making HTTP requests.
- **Glance Widget**: Custom widgets for quick access to app features.
- **Compose Shimmer**: Attractive loading animations using Shimmer effects.
- **Lottie Animation**: Rich animations for enhanced visual appeal.
- **Material 3**: Adopts the latest Material Design guidelines.
- **Hilt**: Dependency injection for managing app components.
- **ArrowKt**: Functional programming library for cleaner and more concise code.
- **Vico**: Charting library for visualizing data trends.

# Features

- **Account Creation**: Users can create a new account by entering their email and password.
  <img src="https://github.com/cisnux-seed/dietary-capstone/assets/68740152/df565abb-0a22-46aa-a4be-8c24276ab801" width="360" height="680">
- **Sign In**: Users can sign in to the app using their registered email and password. They also have the option to sign in using their Google account or link it with their registered email and password.
  <img src="https://github.com/cisnux-seed/dietary-capstone/assets/68740152/ae1373a8-ce08-48c7-b818-b1425006af62" width="360" height="680">
- **User Profile**: Users can add a user profile if they don't already have one, and update their personal data as needed.
  <img src="https://github.com/cisnux-seed/dietary-capstone/assets/68740152/f1237ad7-3e12-4367-9aa8-6de17ca4de3d" width="360" height="680">
- **Food Diary**: The main page features a food diary that users can update daily. Users can filter entries by date and category, search using the search bar, and view the total calorie count of their entries. New entries can be added via a Floating Action Button.\
  <img src="https://github.com/cisnux-seed/dietary-capstone/assets/68740152/e0ff5484-e9b4-4497-b985-908b8cb0859bz" width="360" height="680">
- **Nutritional Needs**: Users receive data on their daily nutritional needs, such as calories, carbohydrates, protein, and fat, which can be viewed in both weekly and daily formats. Users can also monitor changes in their weight and waist circumference.\
  <img src="https://github.com/cisnux-seed/dietary-capstone/assets/68740152/0399c4aa-97cd-4ed8-bd42-a1a7f8327fcb" width="360" height="680">
  <img src="https://github.com/cisnux-seed/dietary-capstone/assets/68740152/5518514d-b750-48e0-8a9f-54b87001f3b5" width="360" height="680">
- **Food Diary Entry**: Users can enter the title and category of their food diary entry.
  <img src="https://github.com/cisnux-seed/dietary-capstone/assets/68740152/7235c318-aabf-4798-9ff2-f61f4695e277" width="360" height="680">
- **Food Recognition**: Users can view the results of the food recognition process, including the name of the food, its contents, feedback, total calories, total carbohydrates, total protein, and total fat. Users can retake the photo or proceed to add the food to their diary.
  <img src="https://github.com/cisnux-seed/dietary-capstone/assets/68740152/5f214b4d-daa3-452b-8fb5-4ef40e5d1513" width="360" height="680">
- **Food Diary Questions**: Users can answer questions about each food item before adding it to their diary. The answers will be used to provide feedback after the food is added to the diary.\
  <img src="https://github.com/cisnux-seed/dietary-capstone/assets/68740152/f1d9c1af-543c-4d4f-924e-75629c57c3be" width="360" height="680">
- **Widget**: Users can utilize the widget to view daily nutrition and the latest food diary entries without opening the app.\
